# 8085-Microprocessor-Simulator

Objective: Design and Develop a desktop Application to emulate the fuctionality of 8085 Microprocessor.

This microprocessor constains following instructions:
A. Load and Store:
1. MOV
2. MVI
3. LXI
4. LDA
5. STA
6. LHLD
7. SHLD
8. STAX
9. XCHG

B. Arithmetic:
1. ADD
2. ADI
3. SUB
4. INR
5. DCR
6. INX
7. DCX
8. DAD
9. SUI

C. Logical:
1. CMA
2. CMP

D. Branching:
1. JMP
2. JC
3. JNC
4. JZ
5. JNZ
